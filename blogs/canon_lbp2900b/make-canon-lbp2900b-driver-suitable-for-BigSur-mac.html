How to install Canon LBP2900/LBP2900b Driver in Catalina/BigSur OS

Download link of patched_driver_lbp2900 as mention below:- https://drive.google.com/file/d/1w6zx...

Location where the patched driver lbp2900 file replace.

  • 1st location - /Library/Printers/Canon/CUPSCAPT2/Bidi/captmoncnab3
  • 2nd location - /Library/Printers/Canon/CUPSCAPT2/cnaccm/CnAC28B9.DAT

 

Useful commends

Make file executable : chmod +x captmoncnab3

Skip the OS security check: xattr -cr /Library/Printers/Canon/CUPSCAPT2/Bidi/captmoncnab3
